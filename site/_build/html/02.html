2. TOAST UI Editor 설치 및 서버 실행¶

2.1. 프로젝트 설치¶

  1. 터미널을 열어 프로젝트를 클론 받습니다.
git clone https://github.com/nhnent/hands-on-labs.toastui.editor-ext.git
  1. npm으로 튜토리얼을 진행에 필요한 개발 환경을 설치합니다.
npm install

2.2. TOAST UI Editor 설치¶

npm을 통해 TOAST UI Editor를 디펜던시로 설치합니다.

npm install tui-editor

2.3. 개발 서버 실행¶

미리 설정된 개발 서버를 실행합니다. 서버 실행이 완료되면 브라우저로 http://localhost:8080/ 에 접속합니다.

npm run serve
